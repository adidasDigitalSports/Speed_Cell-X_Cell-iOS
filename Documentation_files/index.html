This framework is designed to provide support for using Adidas miCoach sensors. In this version, the two supported devices are SPEED_CELL (represented by ADISpeedCellDevice class) and X_CELL (represented by ADIXCellDevice class).

If the serial number of a SPEED_CELL or X_CELL sensor is already known, then the following example demonstrates how to create a ADISpeedCellDevice or a ADIXCellDevice instance:

Create a device instance from serial number

// Create Device Instance Example
// This is an example that shows how to create a device instance from a known serial number
#import <miCoachSensorsUI/miCoachSensorsUI.h>
#import <UIKit/UIKit.h>
@interface CreateDeviceExampleViewController ()
@end
@implementation CreateDeviceExampleViewController
- (NSArray *)createInstancesFromMyKnownSerialNumbers:(NSArray*)serialsWithTypes {
// serialsWithTypes contains NSDictionary object which stores a serial number and a device class string
// Instead of the class string you can use anything to determine the corresponding device type to the serial number
NSMutableArray *devices = [NSMutableArray array];
for (NSDictionary *known in serialsWithTypes) {
NSString *serialNumber = known[@"kSerialNumber"];
NSString *classString = known[@"kClassString"];
ADIGeneralDevice *device;
if ([classString isEqualToString:NSStringFromClass([ADIXCellDevice class])]) {
device = [miCoachSensorsUI xcellDeviceWithSerialNumber:serialNumber];
}
else if ([classString isEqualToString:NSStringFromClass([ADISpeedCellDevice class])]) {
device = [miCoachSensorsUI speedcellDeviceWithSerialNumber:serialNumber];
}
[devices addObject:device];
}
return devices;
}
@end

SDK Licensing

This version of the framework needs online license validation in order to work. Application need to call validation method every time it starts. In case of internet connection not available, but the validation was successful before, app will work for few days, until missed validate will block usage. Important: Please wait until validation finished (completion block called) before using any sensor, otherwise features might not work properly. Here is an example how to do it:

// Authenticate SDK Example
// This is an example that shows how to authenticate your licensed SDK with your username and password.
#import "AppDelegate.h"
#import <miCoachSensors/miCoachSensors.h>
@implementation AppDelegate {
ADILicenseManager *_licenseManager;
}
- (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions {
NSString *clientId = @"<Your client id>";
NSString *clientSecret = @"<Your client secret>";
_licenseManager = [[ADILicenseManager alloc] init];
[_licenseManager validateLicenseWithClientId:clientId clientSecret:clientSecret completion:^(BOOL success) {
if (!success) {
NSLog(@"miCoachSensors framework licensing failed!");
}
}];
return YES;
}
@end
