Detailed Description

ADIBluetoothManager objects are used to manage discovered or connected remote devices.

Method Documentation

- (void) connect: (ADIBluetoothDevice *)  device
completion: (connect_callback_t)  completion 

Connects to the specified remote device represented by an ADIBluetoothDevice object.

The device parameter should be a discovered ADIBluetoothDevice object with valid CBPeripheral.

Parameters
deviceThe device object to connect.
completionBlock object to be executed on completion of the connect. This block has no return value and takes three arguments: a dictionary of CBCharacteristic discovered on the remote device identified by a uuid string from its CBUUID, the connected CBPeripheral and an NSError for possible errors.
- (void) disconnect: (CBPeripheral *)  peripheral
withCallback: (disconnect_callback_t)  callback 

This method disconnects a remote device from the system.

Parameters
peripheralA CBPeripheral object on which connection cancel is called.
callbackBlock object to be executed after the disconnect. This block has no return value and takes a CBPeripheral as argument on which the connnection cancel is called.
- (void) startDiscoveryWithCallback: (discovery_callback_t)  callback

This method initiates a discovery process.

Parameters
callbackBlock object to be executed when a device is successfully discovered. This block has no return value and takes two arguments: advertisement data of the discovered remote device and the CBPeripheral object associated with the remote device.
