Detailed Description

An instance of ADISpeedCellBindingLayer contains the logic of communication with a SpeedCell device.

Method Documentation

- (void) requestANTFS

Requesting remote device to switch to ANT FileSystem mode.

- (void) sendPing

Sending a ping message to remote device used as keep alive.

This message perevents the device from entering sleep state.

- (id<ADIBluetoothDelegateProtocol>) bluetoothDelegate
readwritenonatomicweak

The object that acts as the delegate of the bluetooth protocol.

This delegate is necessary to bet set to allow ADISpeedCellBindingLayer to read and write data on bluetooth.

- (NSUInteger) downloadProgressNotificationLimit
readwritenonatomicassign

A notification limit for download progress. Default is 1.

Every time the progress exceeds this limit sessionDownloadProgressChange: delegate method is called.
