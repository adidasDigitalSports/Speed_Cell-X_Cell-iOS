- (void) downloadDirectory: (void(^)(NSArray *directoryEntries, NSError *error))  completion

This method is responsible for downloading the directory entries from the device.

Parameters
completionA block object to be executed when the download ends. This block has no return value and takes two arguments: an NSArray instance which holds the downloaded ADISpeedCellDirectoryEntry entries and an NSError containing the error. On success the error is nil.
