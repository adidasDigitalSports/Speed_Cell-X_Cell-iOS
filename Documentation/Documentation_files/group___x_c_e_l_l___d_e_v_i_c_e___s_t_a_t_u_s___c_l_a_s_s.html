void xcell_getDeviceStatus ( xcell_context *  context)

This function responsible for getting device status.

The result of this function will be given by a callback as parameter. You can set this callback in the xcell context's didGetDeviceStatus_callback. Possible device status values are declared in c_ADIXCellDeviceStatus enum.

Actual command request bytes will be requested to write out to bluetooth by context's writeToCharacteristics_binding callback.

Parameters
contextA valid non-zero xcell_context instance.

Function requesting device status change.

This function sends out the request to change device status to the requested mode. When reply arrives xcell_handleSetDeviceStatusResponse callback is invoked. Context's requestqueue is resetted on mode change.

Parameters
contextA valid non-zero xcell_context instance.
statusA device status of type c_ADIXCellDeviceStatus.
