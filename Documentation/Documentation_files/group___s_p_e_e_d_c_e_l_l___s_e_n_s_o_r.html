Detailed Description

Basic SpeedCell sensor class.

This is the main class, which is visible from the binding layer.

void sc2_requestDirectory ( sc2sensor_context *  context)

This function initiates a directory download.

This function sends out the command for directory download, which is the first (zero indexed) file on the device. When the process-chain is finished context->didFinishDirectoryDownload_callback is called which takes sc2_directoryEntry objects and number of directory entries.

void sc2_setTime ( sc2sensor_context *  context,
unsigned long  epochTimeInMilliseconds 
)

This function is responsible for setting the given date and time on the device.

This function is asynchron, because it is a file upload operation to the device. When the process-chain is finished context->didFinishSetTime_callback is called which takes a single Boolean argument that indicates whether or not the time set succesfully.

Parameters
contextA valid non-zero sc2sensor_context instance.
epochTimeInMillisecondsEpoch time (a.k.a. POSIX time or Unix time), defined as the number of seconds that have elapsed since 00:00:00 Coordinated Universal Time (UTC), Thursday, 1 January 1970, not counting leap seconds.
