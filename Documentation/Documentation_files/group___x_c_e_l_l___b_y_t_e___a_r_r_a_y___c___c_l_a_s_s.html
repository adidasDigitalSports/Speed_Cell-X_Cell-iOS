Detailed Description

XCell byte array C class.

This array can store unlimited (limited only by available memory) amount of bytes, which means it reallocates itself on overgrow. The array stores the actual given bytes, not pointers to byte arrays. Capacity is the actual capacity of the byte array as bytes in memory, not the number of byte arrays that can be stored.
