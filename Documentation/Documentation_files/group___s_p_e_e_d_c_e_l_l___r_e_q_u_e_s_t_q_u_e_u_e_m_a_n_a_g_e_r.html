Detailed Description

SpeedCell command request queue manager C class.

With this C class you can manage the processing of the queued request in the request queue. It can decide when to pop a request from the queue and write it to bluetooth via writeToCharacteristics_callback.

Enqueues a write request. Does not expect response. Waits until the given time.

This function creates a write request with the given parameters and enqueues it in the context's request queue. This type of write request does not expect any response on the characteristic. It is a high-speed write used for writing out big data. With the parameter blockNextRequestUntil a safe time can be assured, when no requests are sent out.
