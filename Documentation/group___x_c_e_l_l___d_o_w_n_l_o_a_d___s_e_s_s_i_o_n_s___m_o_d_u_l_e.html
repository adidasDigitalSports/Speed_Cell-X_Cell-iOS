Detailed Description

XCell download session module. Handles all the data download functionality.

Function Documentation

void xcell_downloadAllSessions ( xcell_context *  context)

Downloads every session from the device.

This function sends out the command for session download, which is later handled by the appropriate methods. When a session is downloaded didDownloadSession_callback is called. After completion, the session gets erased on the device, and the next session gets downloaded. When every session is downloaded didDownloadSessions_callback is called with the number of sessions.

Parameters
contextA valid non-zero xcell_context instance.

void xcell_readSessionDirectory ( xcell_context *  context)

Initiates a directory download.

This function sends out the command for directory download, which is later handled by the appropriate methods. When the process-chain is finished context->didGetSessionCount_callback is called with the number of sessions. Directory file is stored in context->sessionDirectory.
